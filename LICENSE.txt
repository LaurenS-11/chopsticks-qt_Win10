Chopsticks Qt Game - Classic finger game implementation
Built with Qt5 and C++17 for Windows 10+

Game Rules:
- Players start with one finger up on each hand
- Take turns tapping opponent's hands to add fingers  
- Hands with 5+ fingers are eliminated
- Split even numbers between your own hands
- First player to eliminate both opponent hands wins

Enjoy playing!
